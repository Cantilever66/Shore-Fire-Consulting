# Shore Fire Consulting — Website

Professional fire sprinkler design firm website.

## Tech Stack
- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion (animations)
- Wouter (routing)
- Vite (build tool)

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev
```

## Build for Production

```bash
pnpm build
# Output is in dist/public — deploy to Netlify, Vercel, or Cloudflare Pages
# For Netlify SPA routing, add a _redirects file: /* /index.html 200
```
